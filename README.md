# Active-Directory

## Objective

Build and manage a simulated enterprise network using Active Directory to develop hands-on skills in domain management, user/group administration, Group Policy configuration, security hardening practices, and implementing Role-Based Access Control(RBAC).

## Skills Learned

- Installed and configured Active Directory Domain Services.
- Created and managed users, groups, and organizational units with the use of RBAC to ensure that access to resources is based on user roles within the organization.
- Implemented RBAC by creating security groups and assigning appropriate permissions to restrict access to sensitive information based on user roles.
- Implemented and troubleshooted Group Policy Objects and linked GPOs to specific organizational units.
- Managed DNS and DHCP within a Windows Server environment to support network operations.
- Joined Windows clients to a domain and verified authentication.

## Tools Used

- Vmware workstation pro
- Windows Server 2022
- Powershell
- Active Directory
